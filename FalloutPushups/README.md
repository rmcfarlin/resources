# Vault-Tec Strength Clearance

Printable **8-week push-up protocol** (Fallout light-print theme) aiming at **100 consecutive** push-ups.

## Output

```bash
cd FalloutPushups
npm install   # once
npm run generate
# → Output/Vault-Tec-Strength-Clearance-100.pdf
```

## Pages

1. Cover  
2. Baseline (Day 0) + standing orders  
3–10. Weeks 1–8 (Date / Planned / Actual)  
11. Clearance graduation
